Dragon Ball Characters App 🐉

App desarrollada con React + TypeScript + Tailwind que consume una API pública de Dragon Ball.

Features
- listado de personajes
- paginación
- loader
- hover effects por raza
- lazy loading de imágenes

Tech Stack
- React
- TypeScript
- Vite
- Tailwind

API
https://web.dragonball-api.com/
